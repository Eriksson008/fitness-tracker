# Roadmap

## MVP Complete

- Dashboard-first personal fitness tracker.
- Single local profile.
- BMR/TDEE/calorie/macro targets.
- Weight, nutrition, and workout logging.
- SQLite persistence.
- Docker Compose local deployment.

## Near Term

- Inline edit screens for logged entries.
- Better progress projections: expected goal date, adherence streaks, and weekly averages.
- More useful workout sessions: multiple exercises, per-exercise notes, and previous-performance hints.
- Data export/import as JSON or CSV for local ownership.
- Formal Playwright smoke tests for profile, logging, and dashboard rendering.

## Later

- Adaptive light theme.
- Optional local passcode or browser-only profile guard.
- Charts with trend smoothing and date range filters.
- Progressive Web App support for mobile home-screen use.
- Optional sync target if the local-first model needs a backup story.
